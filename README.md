# Discrete Event Simulator

## Motivation:
Through trying to make predictions on data from a singular production run in a continuous process plant, I quickly realized that the operator's decisions effectively determine output within line. A simple set of heuristic rules and years of knowledge/intuition must guide these engineers. It seems as if they adjust the input variables (e.g. material flow) as they view the incoming data. There are a myriad of factors contributing to poor prediction performance such as learned-distribution change (data drift) due to the machine's burn in period before it reaches its steady state.

The higher value tool lies in simulation. Operators may test their intuition and decisions, observe their custom experiement's outcomes, and find an optimal path without waste in capital.
